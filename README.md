# hong-kong-fintech-career-analyst

This repo is a collection of analyst reports and codes about Hong Kong Fintech Career

## Team member Introduction

### 1. download repo

```bash
# open your cmd and locate any position u want
cd /your/want/position/
git clone git@github.com:ava1anch33/hong-kong-fintech-career-analyst.git
# since you may not have SSH key in your computer, use following command
git clone https://github.com/ava1anch33/hong-kong-fintech-career-analyst.git
cd hong-kong-fintech-career-analyst
```

### 2. modify content and upload to git

u should have the access to modify main branch, so u can simply do this

```bash
cd /your/location/hong-kong-fintech-career-analyst
git add *
git commit -m "your commit message, like: add question description"
git push origin main
```

if u does not have right to modify main branch, do following

```bash
# open your cmd/bash and locate the repo position
cd /your/location/hong-kong-fintech-career-analyst
git checkout -b <your task name>
```

for example, u had added/deleted a file or modified file content like add or remove lines or words

```bash
git add *
git commit -m "your commit message, like: add question description"
git push
```

### 3. update your local repo when other has upload new updated

simply run command git pull and you will see the updates

